# Digital Agency - Responsive React website 

## Features
-**Fully Responsive**: Mobile-first design for all screen sizes.
-**Dark Mode**:Integrated theme toggle using Tailwind's class.
-**Motion Animations**: Smooth scroll-triggered animations and hover effects.
-**Custom Cursor**: Functional contact form powered by Web3Form.

# Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
