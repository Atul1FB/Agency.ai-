# Digital Agency - Responsive React website 

## Features
-**Fully Responsive**: Mobile-first design for all screen sizes.
-**Dark Mode**:Integrated theme toggle using Tailwind's class.
-**Motion Animations**: Smooth scroll-triggered animations and hover effects.
-**Custom Cursor**: An Interactive Mouse Trailer is a smooth "shadow" or circle that follows your cursor with a slight delay. 
                    It makes the website feel fluid and premium.
-**Contact Form**: Functional contact form powered by Web3Form.

# Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
